# 調查局特考 手動匯入資料夾

把你從**考選部考畢試題查詢平臺**下載的官方檔案放這裡。

## 下載哪些檔

1. 到考選部考畢試題查詢平臺：<https://wwwc.moex.gov.tw/main/exam_examreport/wfrmExamReportSearch.aspx>
   （或從考選部首頁 → 應考人專區 → 考畢試題查詢）
2. 搜尋條件選：**調查局特考**、**三等**、**法律實務組**
3. **只下載有「選擇題（測驗題）」的科目**，主要是：
   - 綜合法政知識與英文（憲法、法學緒論、兩岸關係、英文）
   - 國文（測驗部分）
   - （刑法、刑訴、行政法、商事法是申論題，刷題 App 用不到，可略過）
4. 每份試卷下載兩個檔：
   - **試題**（通常是 PDF）
   - **測驗式試題標準答案**（答案，通常是 PDF 或網頁表格）

## 檔名建議

盡量讓檔名含「年份」與「科目」，方便我自動歸類。例如：

```
113_綜合法政知識與英文.pdf          ← 試題
113_綜合法政知識與英文_答案.pdf      ← 答案
114_綜合法政知識與英文.pdf
114_綜合法政知識與英文_答案.pdf
```

名字不完美也沒關係，我會在解析時處理。

## 接下來怎麼跟我配合

1. **先放「一份」試題＋答案當樣本**進這個資料夾。
2. 告訴我「樣本放好了」。
3. 我會檢視真實格式、把解析器調到完全吻合（就像司律一試那樣）。
4. 校準成功後，你再把其餘年份/科目一次放進來，我批次匯入。
